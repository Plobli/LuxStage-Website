# LuxStage Website

Marketing-Website für [LuxStage](https://luxstage.app) — die App für professionelle Beleuchtungsplanung im Theater.

## Inhalt

- `index.html` — Einzel-HTML-Seite (Tailwind CSS via CDN, keine Build-Tools)
- `logo.png` — App-Logo

## Abschnitte

- **Hero** — Headline, Kurzbeschreibung, CTA
- **Apps** — Übersicht iOS-App und Web-App
- **Features** — Funktionsübersicht (Kanalverwaltung, EOS-Import, PDF-Export, Echtzeit-Sync)
- **Workflow** — Schritt-für-Schritt-Ablauf
- **FAQ** — Häufige Fragen
- **Kontakt** — Kontaktformular / -bereich

## Technik

- Reines HTML mit [Tailwind CSS](https://tailwindcss.com) (Play CDN)
- Keine Build-Pipeline, keine Dependencies
- Schriftart: Inter (Google Fonts)
- Akzentfarbe: `#e35d3c`

## Lokale Vorschau

Einfach `index.html` im Browser öffnen oder einen lokalen Server starten:

```bash
npx serve .
```
